## 🛠️ Getting Started

Follow these instructions to set up the project locally and start exploring the blockchain.

### Prerequisites

Ensure you have the following installed:

- [Node.js](https://nodejs.org/) (v16 or higher)
- npm (v8 or higher)

### Installation

1. Clone the repository:

   ```bash
   git clone <YOUR_GIT_URL>
   ```

2. Navigate to the project directory:

   ```bash
   cd bradley-coin-explorer
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm run dev
   ```
